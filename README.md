# 당근북스

## Dev Environment
- Node 18.15.11 (lts)
- Yarn 1.22.18
- Nuxt 3.3.3

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`

```bash
yarn dev
```

## Production

Build the application for production:

```bash
yarn build
```
